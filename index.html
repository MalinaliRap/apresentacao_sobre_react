Bem vindo ao React

  • Versão atual: 16.X.
  • Utiliza o Virtual DOM.
  • O React é baseado em componentes.
  • Foi criado pelo facebook para lidar com o alto trafego de dados de uma página web.
  • Quando o projeto foi liberado em 2013, foi recebido com ceticismo.

Porque as pessoas são céticas quanto ao React?

  • O React é uma pequena biblioteca e não vem com tudo que é necessário para construir uma aplicação.
  • Você escreve algo parecido com HTML (JSX) no seu código javascript.
  • Essas tags precisaram ser pré-processadas antes de rodar no browser.
  • Provavelmente você precisará construir um webpack ou um browserify para isto.

REACT NÃO É UMA FRAMEWORK

  • O React é uma biblioteca para criar interface para o usuário (UI).
  • O React pode ser utilizado no modelo MVC, representando o "V".
  • O React pode ser utilizado com frameworks como Angular, Ember, Redux, Reflux, Flux ...
  • É possivel criar aplicações simples apenas com React

Ferramentas extras para construir aplicações mais robustas:

  • Browserify
  • Webpack
  • React Router
  • React Motion
  • React Addons
  • Enzyme
  • Redux
  • React Fire (Firebase for React)

Components - Componentes

Funcionam como funções JavaScript. Os parametros de entrada são chamados de props e o retorno é dado em elementos HTML (JSX)

State - Estado

O estado pode ser qualquer conjunto de informações que serão utilizadas em algum momento pela interface. Por exemplo: As informações do usuário, uma lista de itens, o resultado de um request ou até informações específicas de UI...

Vamos começar nossa primeira aplicação

Pré-Requisitos

  • JavaScript (De preferência ES6 ou ECMAScript 6), HTML, CSS.

O que é necessário instalar?

  • Node (versão > 6 LTS)
  • npm (versão > 3.x) - incluso na instalação do node.js
  • create-react-app
  • Editor de texto

1. Abra seu terminal windows ou linux

Escolha um diretório. 

    npm install -g create-react-app
    create-react-app nome_da_aplicacao

Comandos 

npm start
npm run build
npm test
npm run eject

2. Delete todos os arquivos do diretório src 


    cd nome_da_aplicacao
    rm -f src/*

Adicione um arquivo CSS chamado index.css no diretório src/ com esse código CSS

Adicione um arquivo JS chamado index.js no diretório src/ com esse código JS

Adicione as três linhas abaixo no começo do arquivo index.js 


    import React from 'react';
    import ReactDOM from 'react-dom';
    import './index.css';
